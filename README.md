# tif_matcher
matches .tif images across two directories


## Requirements:
```
rasterio
```

## Usage:
```
python3 tif_matcher.py [-h] [--dir1 DIR1] [--dir2 DIR2] [--out_file OUT_FILE]
```
* dir1 (str) -      Input directory 1
* dir2 (str) -      Input directory 2
* out_file (str) -  Name of output file
